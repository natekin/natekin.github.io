Competition finale

The main goal of hackathon is semi-supervised Machine Translation: Machine Translation which makes use not only from (often scarce) parallel data, but also from untranslated texts in source and target languages.

Competition prizes

Main prize: journey to ICML 2018 - 35th International Conference on Machine Learning, Stockholm Sweden. Second- and third-best teams will also be awarded valuable prizes.

ML problem

Semi-supervised machine translation for IT-texts from English into German. Evaluation metric is BLEU-4 score: how many words and ngrams overlap in a given translation and a reference translation.

Solution format

Container-based. Here be technical limitations on execution time, container size, memory limits, etc. Also how many solutions a day do we send.

Dataset

This time the topic of our new DeepHack hackathon is machine translation (MT). At the hackathon we will tackle the task of semi-supervised neural MT: we'll try to improve a neural MT model with monolingual data. However, our qualification task is traditional supervised MT It serves for familiarising participants with the MT field.

For this task we have decided to pick a well-studied language pair (English-German) and put no restrictions on the used data and model architecture.
